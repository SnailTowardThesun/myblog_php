## setup

### ubuntu
1. setup php`
  ```
  sudo apt-get update
  sudo apt-get install php7.0 php7.0-fpm php7.0-mysql -y
  ```

1. setup nginx
  ```
  sudo apt-get install nginx`
  ```

1. initialize the environment
  ```
  cd trunk && sh configure 
  ```
